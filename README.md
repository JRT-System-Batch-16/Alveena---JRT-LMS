# JRT LMS — Frontend (React)

A Learning Management System built with React for JRT System.


## Project Structure

```text
jrt-lms/
├── public/
├── src/
│   ├── components/
│   ├── context/
│   ├── pages/
│   ├── styles/
│   ├── App.js
│   └── index.js
├── package.json
└── README.md
```

---

## Getting Started

### Install Dependencies

```bash
npm install

### Run the Project

```bash
npm start

The application will run on:

```text
http://localhost:3000
```
